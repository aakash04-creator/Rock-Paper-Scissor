Stone Paper Scissors Game

This is a simple Stone Paper Scissors game built using basic programming concepts. The project allows a user to play against the computer in a fun and interactive way. The computer randomly selects its move, and the user inputs their choice. Based on the rules of the game, the winner is decided and displayed.

This project helped me understand core concepts like conditional statements, loops, and user input handling. It is beginner-friendly and a great way to practice logic building.

Features
User vs Computer gameplay
Randomized computer choices
Clear win/lose/draw result display
How to Run

Run the program and follow the on-screen instructions to play.
